详细图文教程：添加订阅的7个步骤

下面我们以iOS系统为例，详细介绍Quantumult X添加订阅的全过程：

步骤1：启动Quantumult X应用

在手机主屏找到紫色图标的Quantumult X应用。首次打开时，系统可能会请求VPN配置权限，务必点击"允许"。

步骤2：进入配置界面

点击底部导航栏的"设置"齿轮图标 → 选择"配置"选项。这里是Quantumult X的核心控制中心。

步骤3：访问节点管理

在配置界面右上角找到"节点"按钮。这里会显示当前所有可用的代理节点和订阅列表。

步骤4：添加新订阅

点击右上角的"+"号 → 选择"添加订阅"。此时会弹出URL输入框。

专业技巧：长按输入框可以选择"粘贴"，如果您已经复制了订阅链接，系统会自动识别。

步骤5：输入订阅链接

将事先准备好的订阅链接完整粘贴到输入框中。典型的订阅链接格式类似：
https://example.com/subscribe?token=your_unique_key

重要提示：
- 确保链接完整无遗漏
- 检查是否有特殊字符被自动转换
- 付费服务商的链接通常包含个人识别码，不要随意分享

步骤6：确认并保存

点击"确定"后，Quantumult X会立即尝试连接订阅服务器。如果链接有效，您会看到"订阅添加成功"的提示。

步骤7：首次同步节点

返回节点列表，下拉页面触发手动更新，或点击右上角的刷新图标。稍等片刻，节点列表就会自动填充。

性能优化建议：在Wi-Fi环境下进行首次同步，特别是包含大量节点的订阅可能需要较长时间。

FAQ

SSR 的协议和混淆组合如何选择最适合的？
选择模拟 HTTPS 或 HTTP 的混淆方式，再搭配 AEAD 或协议插件，可以隐藏流量特征并提高抗封锁能力。不同网络环境需测试，选择最稳定和可靠的组合。
WinXray 的多策略组有什么作用？
多策略组可以将节点按用途、延迟或地域分类，结合路由规则实现自动分流。不同类型流量可走不同节点，提升访问速度和稳定性，特别适合多应用使用场景。
Trojan 伪装域名如何选择以提升隐蔽性？
伪装域名应选择真实可访问网站并配备有效证书，这样 Trojan 流量可模拟正常 HTTPS 请求，降低被封锁和检测的风险，提高节点长期可用性和访问稳定性。
V2Ray 如何使用 WebSocket 传输？
在 V2Ray 配置文件中将传输方式设置为 ws，即启用 WebSocket 模式。结合 CDN 和 TLS，可实现安全高速的代理链路。WebSocket 常用于伪装成 HTTPS 流量，从而降低被识别风险。
Nekobox 导入 Clash 订阅后规则丢失，可能原因？
部分客户端在解析订阅时只保留节点信息而忽略 rules 或 proxies-group，建议使用支持完整解析的订阅转换器（如 sub-converter）先将订阅转换为兼容格式再导入，或手动合并规则文件。
Vmess 的安全加密方式推荐使用哪种？
目前推荐使用 auto 或 none（结合 TLS 加密）以提升性能。旧版 AEAD 加密已被废弃，使用 TLS/XTLS 层提供的加密即可确保安全。过度加密反而增加 CPU 占用与延迟。
如何优化 Clash 的 DNS 设置？
在 Clash 中使用自定义 DNS 可以加快域名解析速度。建议使用 doh（DNS over HTTPS）或 dot（DNS over TLS）配置方式，既能提升速度，也能保护隐私防止 DNS 泄露。
Hiddify Next 订阅更新后节点消失，如何恢复？
先检查订阅源是否更改格式或被下架，尝试重新粘贴订阅链接并手动刷新；若仍缺失，联系订阅提供方确认是否停用某些节点或使用了新的访问权限机制（例如 token 验证）。
Mihomo 支持节点分类管理吗？
支持。用户可将节点按地区、用途或延迟分类，结合策略组和规则进行分流。这样可以快速选择最佳节点，提高访问效率并便于维护大量节点。
V2Ray 策略组如何根据节点延迟自动选择？
策略组会根据节点延迟和丢包率自动选择最优节点。结合路由规则可保证国内流量直连，海外流量走最佳节点，提高访问速度和连接稳定性，优化整体网络体验。
